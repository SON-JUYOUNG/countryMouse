# LCI Frontend Client

## Overview
React 19 application built with Vite and Tailwind CSS 4.
Visualizes LCI data on Kakao Map and provides simulation UI.

## Quick Start
1.  Setup environment variables (see `.env.example`).
2.  Install and Run:
```bash
npm install
npm run dev
```

## Stack
*   React 19
*   Vite
*   Tailwind CSS 4
*   Zustand
*   Kakao Map SDK
