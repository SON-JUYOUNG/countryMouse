# LCI Backend Service

## Overview
Based on Spring Boot 4.0 and Java 25.
Provides REST APIs for Life Cost Index (LCI) data and financial simulations.

## Quick Start
```bash
./gradlew bootRun
```
*   **API Docs**: `http://localhost:8080/swagger-ui/index.html`
*   **Health Check**: `http://localhost:8080/api/health`

## Stack
*   Java 25 (LTS)
*   Spring Boot 4.0.0
*   H2 Database
*   Spring Security
